### My own brainf__k interpreter, written in node.

Uploaded this to NPM just for the sake of convenience.

---

Usage:
```js
// CommonJS
const { run } = await import('bfscript');

// ESM
import { run } from 'bfscript';


run('>++++++++[<+++++++++>-]<.>++++[<+++++++>-]<+.+++++++..+++.>>++++++[<+++++++>-]<++.------------.>++++++[<+++++++++>-]<+.<.+++.------.--------.>>>++++[<++++++++>-]<+.');
```
